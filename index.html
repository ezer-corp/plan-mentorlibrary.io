🔄 화면 흐름도 (Navigation Flow)
[공통 GNB] ─────────────────────────────────────────────
  로고 클릭          → SCR-001 메인
  서비스 ▾           → 드롭다운 (SCR-002~005)
  멘토단             → SCR-006
  고객 사례          → SCR-007
  문의               → SCR-008
  무료 상담 신청      → SCR-008

SCR-001 메인 (/)
  ├─ "기업 서비스 알아보기"    → SCR-002 리더십과 전략 (첫 번째)
  ├─ "무료 상담 신청"          → SCR-008 문의
  ├─ 서비스 카드 "자세히 보기" → SCR-002 / SCR-003 / SCR-004 / SCR-005
  ├─ 멘토 카드 클릭            → SCR-006 멘토단
  └─ 하단 CTA 버튼             → SCR-008 문의

SCR-002 ~ SCR-005 서비스 페이지 (/service/*)
  ├─ 서비스 탭 클릭            → 해당 서비스 페이지 전환 (동일 URL 경로 내)
  ├─ 관련 멘토 카드 클릭       → SCR-006 멘토단
  └─ 서비스별 CTA 버튼         → SCR-008 문의

SCR-006 멘토단 (/mentors)
  ├─ 멘토 카드 클릭            → 멘토 상세 모달 (인라인 팝업)
  ├─ "멘토 지원하기"           → 멘토 지원 폼 (별도 구현 또는 문의폼 활용)
  └─ "기업 도입 문의"          → SCR-008 문의

SCR-007 고객 사례 (/cases)
  ├─ 사례 카드 클릭            → 사례 상세 (인라인 팝업 또는 /cases/{id})
  └─ 하단 CTA 버튼             → SCR-008 문의

SCR-008 문의 (/contact)
  └─ 폼 제출 완료              → 완료 메시지 (인라인) + 자동 이메일 발송
※ 아임웹 특성 상 드롭다운 메뉴, 탭 전환, 팝업은 아임웹 내장 기능으로 구현. 별도 커스텀 코딩 최소화
